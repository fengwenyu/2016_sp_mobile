一、挑选商品

找到自己喜爱的商品，点击“购买”按钮，商品会添加到您的购物袋中。

温馨提示：

购物袋中的商品无法保留库存，请您及时结算，商品库存以订单提交时为准。

二、确认下单

1. 在我的购物袋中，点击‘去结算’

2. 确认订单信息，提交订单

3. 选择支付方式，去支付

温馨提示：

如果您在支付过程中超过30分钟，订单会自动取消；如需保留订单，请联系客服。

4. 提交订单成功

三、订单跟踪

在等待收货过程中，您可以在我的订单中查询订单状态，了解订单处理及商品配送进度。

四、验货签收

请您在配送人员在场情况下，当面开包验收您的商品。

1. 为了保障您的购物权益，请务必仔细检查包装是否完好（封条无破损、补封），并与快递人员当面验收货品，快递人员离开前相关问题均由我司负责。如当场发现有任何问题，请当面拒收并致电：尚品客服电话（4006-900-900）

2. 如您已签收确认，尚品网均视为您已对产品的外观、品质、产地、颜色、尺寸、材料、包装等非客户主观方面表示认同，且不会以主观原因为理由进行退换货，因此，请您务必认真验货！

3. 签收商品后，如果商品出现质量问题，请与尚品客服中心（4006-900-900）联系。

4. 如他人代签收且无异议，尚品网均认定为本人签收且同意！

其他说明：

1. 发货时间：我们在核对会员订购的商品、邮寄地址、款项支付等信息无误后确认订单，订单确认后货物会及时发出。

2. 收件人请填写中文名字，如果使用非中文名字，请务必与身份证或护照保持完全一致，以保证您的包裹可以准确送达。如因此造成包裹无法送达或丢失，尚品不承担任何责任。

3. 配送时间以各地区预计配送时间为准，订单发货当天不计算在配送时间内。

特殊说明：

1. 改寄：发货前改变邮寄地址不另收费，发货后恕不受理；

2. 签收：请检查包裹有无破损，如无问题，请在物流公司指定单据上签收，物流公司会将签收信息返回尚品网。

3. 拒收：请检查包裹，如有明显损坏迹象，您可以拒收该包裹，并及时通知我们，我们会与物流公司协商赔偿事项，并承担由此而产生的配送费用；因会员主观原因拒收，配送费用由会员承担。

请您确保订单中的收货地址、邮编、电话、Email地址等各项信息的准确性，以便商品及时、准确地发出，并随时查看个人账户里“订单中心”的订单状态更新情况，如果因为以上信息有误，我们会把订单审核为不合格，不合格订单会放入“异常终止订单”区处于锁定状态，会员可以根据提示并修改后再次提交订单。
